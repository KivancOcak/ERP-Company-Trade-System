# ERP-Company-Trade-System
A new browser System that Allows user to Register Login and Apply all kind of ERP System Actions with High Developed UI Navigation System that Contains Virtual Maps Graphs Purchase Sale Systems 

HOW TO IMPORT AND RUN INTO PC
Project is Created by HTML CSS and Java Script Languages

After Download All Neccessary Files inside Visual Studio Code Add VS Code Extensions called Live Server in order to Run System Automaticly without any manual Web Port Definition After Live Server İnstallation Complete Click on Go live button from VS Code Menu Screen and System will start Run and output will be displayed
